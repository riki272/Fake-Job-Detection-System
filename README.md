🚨 AI Fake Job Detection System

AI-powered web application that detects **fraudulent job postings** using Machine Learning and Natural Language Processing.

It analyzes job descriptions and predicts whether a posting is **Real ✅ or Fake ⚠**, helping job seekers avoid scams.

---

🌐 Live Demo

🔗 Deployed App: **https://riki272-fake-job-detection-system-app-vdgeze.streamlit.app/**

---

⚡ Key Features

* Analyze any job description instantly
* ML-based classification (Real vs Fake)
* Confidence score for prediction
* Reset functionality
* Interactive web app interface
